# Simple Calculator (C# Console Application)

## Description
This is a basic calculator program written in C#. It allows the user to:
- Enter two numbers
- Choose an operator: `+`, `-`, `*`, or `/`
- Displays the result

## Features
- Input validation for division by zero
- Clear and simple output
